Para crar el entorno virtual: python -m venv venv

Para activarlo: source venv/Scripts/activate

Para crear el proyecto: reflex init

Para correr el proyecto: reflex run 

Para actualizar reflex: pip install --upgrade reflex

Para loguearme en el Hosting de reflex: reflex login

Para desplegar la pagina: reflex deploy

Para bajar la pagina: reflex deployments delete blog-personal
